# Kova IDE

Build native. Ship fast.

Kova is a proprietary AI-powered Integrated Development Environment (IDE) built specifically for developers who build native desktop applications.

## Quick Start

1. Install dependencies: `pnpm install`
2. Run backend: `pnpm --filter kova-backend dev`
3. Run IDE: `pnpm --filter vscode run watch`

For complete AI Rules & Architecture, check `docs/AI-MASTER-DOC.md`.
